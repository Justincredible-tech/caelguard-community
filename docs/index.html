ShellGuard Scanner

Free / Open Source

Three-tier threat detection for OpenClaw skills. Catches prompt injection, obfuscated payloads, credential exfiltration, and tool abuse patterns that VirusTotal misses.

  • 3-tier regex engine: literal, obfuscation, behavioral
  • Suspicion index 0-100 with severity breakdown
  • Scans SKILL.md + all referenced scripts
  • --all-installed: audit every skill in one command
  • JSON output for CI/CD integration
