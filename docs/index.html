ShellGuard Scanner

Free / Open Source

Three-tier threat detection for OpenClaw skills. Catches prompt injection, obfuscated payloads, credential exfiltration, and tool abuse patterns that VirusTotal misses.

  • 3-tier regex engine: literal patterns, obfuscation detection, behavioral analysis
  • Suspicion index 0-100 with per-finding severity breakdown
  • Scans SKILL.md + all referenced scripts and configs
  • --all-installed mode: audit every skill on your system in one command
  • JSON output for CI/CD integration
python3 shellguard-scanner.py --all-installed

Shadow Detector

Free / Open Source

Cross-skill shadowing analysis. Detects when multiple skills try to override the same tools, create conflicting instructions, or establish hidden dependencies. Our killer differentiator -- nobody else does this.

  • Detects tool shadowing (two skills claiming the same command)
  • Identifies instruction conflicts and priority manipulation
  • Maps hidden cross-skill dependencies
  • Alerts on skills that modify other skills' behavior
python3 shadow-detector.py /path/to/skills/

Token Audit

Free / Open Source

Workspace token analysis and cost estimation. Scans your entire workspace, estimates token consumption across 6 major models, and generates optimization recommendations.

  • Scans all workspace files with smart exclusions
  • Cost estimates for GPT-4o, Claude Opus/Sonnet, Gemini Pro, DeepSeek
  • Identifies largest token consumers and optimization targets
  • Markdown report generation
python3 token-audit.py ~/your-workspace/
